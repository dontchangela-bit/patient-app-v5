# AI-CARE Lung - 病人端

## 部署步驟

1. 在 GitHub 建立新 Repo（例如：aicare-patient）
2. 上傳這個資料夾內的所有檔案
3. 到 Streamlit Cloud 部署
4. Main file 選擇 `app.py`

## 檔案清單
- app.py（主程式）
- config.py（設定，請填入 API Key）
- data_manager.py（資料管理）
- requirements.txt（套件）
- data/patient_records.json（資料儲存）
- .streamlit/config.toml（樣式設定）

## 設定 API Key
編輯 config.py，填入您的 OpenAI API Key
